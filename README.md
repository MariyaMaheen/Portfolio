# 🌐 Mariya Maheen — Portfolio Website

A premium, minimal, and interactive personal portfolio website showcasing my work as a **Software Engineer & Data Analyst**.  

Designed with a strong focus on **aesthetics, performance, and user experience**.

---

## ✨ Features

### 🎯 Modern Luxury UI
- Dark theme with gold accents  
- Elegant typography (Cormorant + Syne)  
- Glassmorphism-inspired elements  

### 🖱️ Custom Cursor System
- Smooth cursor tracking  
- Interactive hover animations  
- Expanding cursor ring effect  

### 🔦 Flashlight Effect
- Dynamic radial spotlight following the cursor  
- Creates depth and immersive interaction  

### 🎬 Scroll Reveal Animations
- Built using Intersection Observer  
- Smooth fade-up transitions on scroll  

### 📱 Responsive Design
- Fully optimized for desktop and mobile  
- Adaptive grid layouts  

### ⚡ High Performance
- Built with **Vanilla HTML, CSS, JavaScript**  
- No external frameworks  
- Lightweight and optimized
